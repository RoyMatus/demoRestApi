# Demo REST API
Demo REST API is a personal playground 
to learn different programming technologies.

## Installation
You need:
- Java 17
- Docker
- Postgres

```bash
Get-ChildItem Env:
```

## Usage
Get all records from database: GET http://localhost:8080/records
